# # Loan Default Risk Analysis Dashboard

###  Project Overview
This Tableau dashboard explores borrower behavior and identifies key factors driving loan defaults.  

###  Tools Used
- Tableau Desktop (Public Edition)
- Excel / CSV dataset

### Key Insights
- Lower credit scores and younger borrowers show higher default risk.
- Income stability and employment type strongly correlate with repayment.
- Home and Education loans perform better than Business loans.
